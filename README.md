# 탄소 배출 관리 시스템 (Carbon Emission Management System)

## 📌 프로젝트 개요
기업 활동 데이터(전기, 원소재, 운송 등)를 입력하면  
배출계수 기반으로 탄소 배출량(kgCO2e)을 자동 계산하고 시각적으로 관리할 수 있는 웹 애플리케이션입니다.

사용자는 데이터를 입력하고, 카테고리별 배출량과 총 배출량을 직관적으로 확인할 수 있습니다.

---

## 🛠 기술 스택
- Frontend: React, TypeScript, Vite
- State Management: useState, useMemo
- Version Control: Git, GitHub

---

## 🚀 주요 기능

### 1. 활동 데이터 입력
- 날짜, 활동 유형(전기/원소재/운송), 설명, 수량 입력
- 입력값 validation 처리 (빈 값 및 0 이하 값 방지)

### 2. 탄소 배출량 자동 계산
- 활동 유형별 배출계수 적용


### 3. 카테고리별 요약
- 전기 / 원소재 / 운송 각각의 배출량 표시
- 총 배출량 계산 및 강조 표시

### 4. 데이터 테이블 관리
- 입력된 데이터 리스트 표시
- 개별 데이터 삭제 기능

### 5. 샘플 데이터 기능
- 버튼 클릭으로 테스트 데이터 자동 입력
- 검토자가 빠르게 기능 확인 가능

## 🧠 설계 의도 및 판단

### 1. 상태 관리 구조
- 모든 데이터는 activities 배열로 통합 관리
- 계산 로직은 useMemo를 활용해 성능 최적화

👉 이유: 데이터 변경 시 자동으로 결과 갱신되도록 설계

---

### 2. 배출량 계산 로직 분리
- calculateEmission 함수로 분리

👉 이유:
- 재사용성 확보
- 유지보수 용이성

---

### 3. UI 구조 설계
- 입력 / 결과 / 요약 영역 분리

👉 이유:
- 사용자가 데이터 흐름을 직관적으로 이해 가능
- 실제 SaaS 형태 UI 구조 참고
